🚀 COMO EXECUTAR O SITE:

1. MÉTODO RÁPIDO - PYTHON (Recomendado):
   - Clique duas vezes em "iniciar-servidor.bat"
   - O navegador abrirá automaticamente
   - O PDF da Mesa Posta funcionará perfeitamente!

2. MÉTODO RÁPIDO - NODE.JS (Alternativo):
   - Clique duas vezes em "iniciar-servidor-node.bat"
   - O navegador abrirá automaticamente
   - Funciona se Python não estiver instalado

3. MÉTODO MANUAL - PYTHON:
   - Abra o terminal/prompt na pasta do projeto
   - Digite: python server.py
   - Acesse: http://localhost:8000

4. MÉTODO MANUAL - NODE.JS:
   - Abra o terminal/prompt na pasta do projeto
   - Digite: node server.js
   - Acesse: http://localhost:8000

5. MÉTODO DIRETO (Pode não carregar PDFs):
   - Clique duas vezes em "index.html"
   - Funciona, mas PDFs podem não carregar
